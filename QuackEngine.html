Quack Engine

Quack Engine is a 3D data oriented engine that as a group of nine develop during our third year of collage using Ogre3D for the rendering. The core engine is programmed in C++ while the data oriented objects are programmed using Lua. The sound system is made using FMOD, the phisics with Bullet and the UI with CEGUI.

Using the Quack Engine engine, we made two diferent show games, in my case, I worked on Splash Showdown, a simple 1v1 top down tank game using a watery enviroment. This game is a showcase of what can be done with our custom engine.

In both of the projects we all worked over all areas, but in the engine I focused more in the integration and the creation an abstraction layer for FMOD, CEGUI and the system of the lecture of the Lua objects from C++. In Splash Showdown, I worked mainlly on the assets, menus, and creation and design of the maps.

The code for both the engine and the game is public on Github. Engine GitHub. Splash Showdown GitHub
