Programa de Convivencia Espacial

Programa de Convivencia Espacial is 2D local multiplayer party-game. Players must compete each other in a zero gravity environment during short but eventful rounds. Players can move jumping from objects or using special objects situated around the map. There is a variety of maps and objects to play with over the different gamemodes.

The game is made using C++ from scratch using a Entity Component Systen with a State Machine for the different modes and menus. In addition, there is a particle system and an animation system for visual effects. For the graphics we used SDL and Box2D for the physics.

We were a team of 10 people in total working for five months. It was our first big project starting from scratch and the main focus was to experience the workflow of making an actual videogame. Therefore, every member of the group touched nearly every aspect as the game expanding from the UI, to core mechanics, to the base of the "engine".

In general, I'm actually really proud of what we managed to achieved. Just as we started the project, we all got into lockdown and taking into account that it was our first project, I find that ended up with a decent project that I will always remember. The game can be downloaded on our itch.io and the code accessible on GitHub. In addition, we also have a trailer on YouTube
